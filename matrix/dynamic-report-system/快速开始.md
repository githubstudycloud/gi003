# 快速开始指南

## 第一步: 打开系统

在浏览器中打开 `index.html` 文件

## 第二步: 体验示例

### 快速生成混淆矩阵报告

1. **填写基本信息**
   - 报告名称: `测试报告`
   - 版本名称: `v1.0`

2. **选择报告类型**
   - 选择 "分类预测混淆矩阵统计报告"

3. **加载示例数据**
   - 点击 "🎲 使用示例数据" 按钮
   - 系统会自动生成200条数据

4. **配置字段映射**
   - 预期值字段: 选择 `expected_level`
   - 实际值字段: 选择 `actual_level`
   - 实际值显示说明字段: 选择 `level_name`

5. **设置策略**
   - 矩阵最值策略: 保持默认"自动计算"
   - 空值处理策略: 保持默认"记为0"

6. **生成报告**
   - 点击 "🚀 生成报告" 按钮
   - 自动跳转到混淆矩阵报告页面

## 第三步: 查看报告

### 混淆矩阵报告页面功能

1. **查看统计信息**
   - 顶部卡片显示: 总样本数、预测正确数、预测错误数、总准确率

2. **分析混淆矩阵**
   - 绿色单元格: 预测正确
   - 黄色单元格: 预测错误
   - 灰色单元格: 无数据
   - 左侧第一列: 显示实际值的说明文字

3. **查看详细数据**
   - 点击任意非零单元格
   - 弹出窗口显示该单元格的所有原始数据

4. **查看指标分析**
   - 每行右侧: 召回率
   - 每列底部: 精准率
   - 颜色表示:
     - 绿色: ≥90% (优秀)
     - 橙色: 70-90% (良好)
     - 红色: <70% (需改进)

5. **返回主页**
   - 点击左上角 "← 返回主页" 按钮

## 第四步: 使用自己的数据

### 准备JSON数据文件

创建一个JSON文件,格式如下:

```json
[
  {
    "预期等级": 3,
    "实际等级": 3,
    "等级说明": "中等",
    "场景": "推荐系统",
    "备注": "测试数据"
  },
  {
    "预期等级": 5,
    "实际等级": 4,
    "等级说明": "较高",
    "场景": "搜索排序",
    "备注": "测试数据"
  }
]
```

### 上传并配置

1. 返回主页
2. 点击 "📁 选择JSON文件" 上传你的数据
3. 在字段选择器中:
   - 预期值字段: 选择 `预期等级`
   - 实际值字段: 选择 `实际等级`
   - 显示说明字段: 选择 `等级说明`
4. 生成报告

## 常见问题

### Q1: 字段选择器为空?
**A:** 请先上传JSON文件或点击"使用示例数据"

### Q2: 生成报告时提示"字段不能重复选择"?
**A:** 预期值、实际值、显示说明必须选择不同的字段

### Q3: 报告页面显示"未找到数据"?
**A:** 确保在主页面成功加载了数据后再生成报告

### Q4: 混淆矩阵太大显示不完整?
**A:**
- 使用"手动设置"矩阵最值策略,设置较小的值
- 或者调整浏览器缩放比例

### Q5: 数据中有空值怎么办?
**A:** 在"空值处理策略"中选择:
- "记为0": 如果0是合理的值
- "抛弃": 如果希望忽略这些数据
- "报异常": 如果需要严格的数据验证

## 高级技巧

### 1. 自定义矩阵大小
如果你知道数据范围是0-7,可以:
- 选择"手动设置"
- 输入 8 (因为包含0,所以是8x8矩阵)
- 这样可以避免矩阵过大

### 2. 数据质量检查
- 选择"报异常"策略
- 如果有空值或异常数据,系统会明确提示
- 修复数据后重新上传

### 3. 查看详细数据
- 点击混淆矩阵中的任意单元格
- 可以看到该预测结果的所有原始数据
- 便于分析预测错误的原因
